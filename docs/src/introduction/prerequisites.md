# Prerequisites

## Dependencies

> **[Note]** `tauri-fuzz` only works with Tauri 2.x.

- [Rust](https://www.rust-lang.org/tools/install)
- [LibAFL](https://github.com/AFLplusplus/LibAFL)
- [Tauri](https://tauri.app/start/prerequisites/)
